# Student-Database-Manager
Simple Student Management System in C with file handling. It allows the users to add, view, search and delete student records which are stored in a text file.


# 📚 Student Management System [C language Project]

A simple C based Student Management System with file handling in console.  
The project allows you to handle the student record in an efficient way using menu driven program.

---

## Features
- ➕ New student
- 📋 show all students
🔍 Find student by id
- ❌ Delete student by ID
- 💾 Data stored as text file (`Aymane.txt`)

---

## 🛠️ Technology Used
- Language C
- File Handling (fopen, fprintf, fscanf)
- Buildings

---

## 📂 The way it works

All student data is stored in a text file with this format:

ID|Name|Age|Grade
